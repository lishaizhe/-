如果有些东西,比如地表不需要在游戏中静态计算，我们可以在编辑模式下将其改成static模式

有下面一些参数可以选择

Nothing   ------------------  对任何系统都不使用静态计算(预计算)
Everything  ----------------  所有情况都使用
Contribute GI  -------------  在预计算光照数据时包含游戏对象
Occluder Static ------------  在遮挡剔除系统中，将游戏对象标记为静态遮挡物
Occludee Static  -----------  在遮挡剔除系统中，将游戏对象标记为静态被遮挡物
Batching Static ------------  将游戏对象的网格与其他符合条件的网格组合起来，有可能降低运行时渲染成本
Navigation Static ----------  在预计算导航数据时包含游戏对象
Off Mesh Link Generation  --  在预计算导航数据时，尝试生成一个从该游戏对象开始的网格外链接
Reflection Probe  ----------  为 Type 属性设置为 Baked 的光照探针预计算数据时，包含此游戏对象
